## How to test

1. Clone the repository
2. Create your own `KEY_API` from [themoviedb](https://developers.themoviedb.org/)
3. Add at the root of the folder `env.js` file :

```js
export const THEMOVIEDB_API_KEY = 'YOUR_API_KEY'
```

4. Have fun üëå
